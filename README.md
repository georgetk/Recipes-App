Run:
```yarn install && npx pod-install && yarn ios && yarn android```

Highlights:
- ```zustand``` and ```@react-native-async-storage/async-storage``` for recipe saving
- ```@tanstack/react-query``` for API response caching
- ```@shopify/flash-list``` for optimized list performance
- ```react-native-keyboard-controller``` for KeyboardAwareScrollView
- Various custom hooks for reusable logic across components
- Advanced TypeScript usage
- Tests using ```jest``` and ```@testing-library/react-native```

API integrations using https://www.themealdb.com/ :
- For trending recipes
- For categories
- For recipes based on category
- For recipe details
- For random recipe
